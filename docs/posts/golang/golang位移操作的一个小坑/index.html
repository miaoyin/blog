问题描述

现有int16类型的整数，将它转为两个字节存放。在还原时，一不小心会就被坑了。

错误用法

length := int(keyBytes[cursor] + keyBytes[cursor+1] <<8)

正确用法

keyBytes := []byte{ 16, 1}
length := int(uint16(keyBytes[cursor]) + uint16(keyBytes[cursor+1]) <<8)

错误原因

字节byte位移时类型不会自动变化, 先要转换类型再位移。
