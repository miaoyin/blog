原因

tensorflow在为张量shape[64,33,33,2048]分配gpu内存是发现资源不够。

假如数据类型是int8, 该张量需要的内存大小64333320481B=142737408B = 142.7MB

解决方法

  1. 降低图片质量
  2. batch_size改成1
  3. 改用大内存的显卡
  4. 增加显卡, 并行训练
